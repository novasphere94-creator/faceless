Privacy Policy

This Privacy Policy describes how Faceless Automation handles data when interacting with social media APIs such as TikTok, Instagram and YouTube.

1. Data We Collect

  • OAuth access tokens (required for authentication)
  • Public profile information provided by the user during login
  • Uploaded media files needed for posting content

2. How We Use Data

Data is used strictly for publishing videos on behalf of the user. No data is transferred or sold to third parties.

3. Data Storage

Media files are temporarily processed for uploading and can be deleted at any time. Tokens are stored securely and used only for API actions.

4. User Control

Users may revoke access anytime through TikTok settings. All tokens become invalid immediately.
